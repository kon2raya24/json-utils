# json-utils

[![npm version](https://img.shields.io/npm/v/json-utils.svg)](https://www.npmjs.com/package/json-utils)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Safe JSON parse, stringify, and patch utilities

## Installation

```bash
npm install json-utils
```

## Functions

- `safeJsonParse` — Parse JSON safely without throwing
- `safeJsonStringify` — Stringify with circular ref handling
- `jsonPatch` — Apply a simple key-value patch to a JSON object
